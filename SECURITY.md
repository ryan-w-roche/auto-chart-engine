# Security Policy

## Supported Versions

Only the latest release of Auto Chart Engine receives security updates.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability, report it privately through one of the following:

- **GitHub Private Vulnerability Reporting** — open a [Security Advisory](https://github.com/ryan-w-roche/auto-chart-engine/security/advisories/new) directly on this repository
- **Email** — contact the maintainer directly at the email listed on their GitHub profile

### What to include

To help us triage and resolve the issue quickly, please provide:

- A description of the vulnerability and its potential impact
- Step-by-step instructions to reproduce it
- Any relevant code, logs, or screenshots
- Your suggested fix, if you have one (optional)
